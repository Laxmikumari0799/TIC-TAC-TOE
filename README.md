This is a simple web-based version of the popular two-player game, built with HTML, CSS, and JavaScript.

## Features
- Playable between two users on the same device.
- Automatic winner detection.
- Game resets after each round.
- Clean and intuitive UI.

## Technologies Used
- **HTML5**: Structure of the game board.
- **CSS3**: Styling the game board and elements.
- **JavaScript**: Game logic, player moves, and winner determination.

## How to Play
1. Open the `index.html` file in a web browser.
2. Player 1 uses "X" and Player 2 uses "O".
3. Click on the grid to place your mark.
4. The game automatically checks for a winner and resets for the next round.

## How to Run
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the `index.html` file in any modern web browser.
